# Random Task Generator

**Автор:** Баданин Максим

## Описание
Приложение помогает бороться с прокрастинацией, выбирая для пользователя случайную полезную задачу из списка. Все сгенерированные задачи сохраняются в историю.

## Функционал
- Выбор категории задач (Учёба, Спорт, Работа).
- Генерация случайного задания с помощью библиотеки `random`.
- Просмотр истории заданий.
- Фильтрация истории по типу активности.
- Сохранение данных в JSON.

## Как использовать
1. Запустите файл `task_generator.py`.
2. Выберите интересующую вас категорию в выпадающем списке.
3. Нажмите кнопку «Сгенерировать задачу».
4. Полученная задача отобразится на экране и добавится в список истории внизу.
5. Чтобы посмотреть только спортивные или учебные задачи, выберите нужный пункт в фильтре и нажмите «Применить».

## Тестирование
- **Проверка случайности:** При многократном нажатии в одной категории задачи выбираются в случайном порядке.
- **Проверка сохранения:** Закройте программу и откройте снова — история ваших задач загрузится автоматически из файла `task_history.json`.
