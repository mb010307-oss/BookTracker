# Expense Tracker

**Автор:** Баданин Максим

## Описание
Программа для учета личных финансов. Позволяет добавлять расходы, указывать категории и даты, а также автоматически сохраняет данные в JSON. Реализован графический интерфейс (GUI) на базе библиотеки Tkinter.

## Функции
- Добавление записей о расходах (сумма, категория, дата).
- Валидация ввода (проверка корректности суммы и формата даты).
- Автоматический подсчет итоговой суммы.
- Хранение данных в формате JSON для доступа после перезапуска.

## Как использовать
1. Убедитесь, что у вас установлен Python 3.x.
2. Скачайте файлы репозитория.
3. Запустите программу через терминал: `python main.py`.
4. Введите сумму расхода, выберите подходящую категорию и укажите дату в формате ГГГГ-ММ-ДД.
5. Нажмите кнопку "Добавить расход", чтобы обновить таблицу и итоговый баланс.

## Тестирование
- **Проверка суммы:** Если ввести отрицательное число или текст в поле "Сумма", программа выдаст предупреждение об ошибке.
- **Проверка даты:** Если формат даты отличается от ГГГГ-ММ-ДД (например, 31.12.2023), программа потребует ввести корректную дату для сохранения записи.
