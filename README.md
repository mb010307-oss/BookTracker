# Weather Diary (Дневник погоды)

**Автор:** Баданин Максим

## Описание
Приложение для ведения ежедневного дневника погодных условий. Позволяет сохранять историю температур, описаний и осадков в удобном формате.

## Возможности
- Добавление записей с автоматической проверкой формата даты и температуры.
- Фильтрация записей: отображение погоды выше указанной температуры.
- Автоматическое сохранение данных в файл `weather_data.json`.
- Графический интерфейс для удобного просмотра всей истории наблюдений.

## Как запустить
1. Установите Python 3.
2. Скачайте файл `weather_main.py`.
3. Запустите его командой:
   ```bash
   python weather_main.py
